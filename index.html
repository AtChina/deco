    Azure Cross-Platform Storage Explorer

    A cross-platform implementation of azure storage explorer allowing the manipulation of azure blob storage accounts and eventually tables and queues.

    If you've ever used Azure Blob storage, you'll notice that there really isn't a blob explorer which will let you browse your assets easily. This project aims to make it easier for you to browse Azure blob storage without having to write a program on your Mac, Linux, or Windows box.

    Features

    • Create Containers
    • Search Containers
    • Upload to Containers
    • Bulk download of blobs
    • Preview media blobs such as images, audio and video
