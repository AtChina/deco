Manage Your Storage Accounts like a Pro

Microsoft Azure Storage Explorer (Preview) is a standalone tool to easily work with your Azure Storage Accounts. This preview release enables you to easily work with Blob Storage: create and delete containers; upload, download, and delete blobs, and search across all your containers and your blobs. For developers and ops, there are more advanced issues to work with SAS keys and Policies. With the free Storage Explorer, you’re in full control of your assets.

Features

  • Generate SAS keys
  • Manage and create Stored Access Policies
  • Search for blobs by prefix
  • Drag ‘n drop files to upload or download

Known Issues

Click to see the full list of known issues

  • Cannot view/take actions on Queues or Tables besides creation and deletion
  • After reentering credentials, you may need to manually refresh Storage Explorer to show your Storage Account
  • Activity log entries may sometimes remain in an "in progress" state when an error occurs, and the error is not displayed
  • When setting blob container public access level, the new value is not updated until you re-set the focus on the container. Also, the dialog always defaults to "No public access", and not the actual current value.
  • Cannot rename blobs from blob download dialog
  • Uploading or download a large number of blobs (500+) may sometimes cause the app to have a white screen
  • No keyboard shortcut to delete blobs - must use delete action on toolbar or context menu command
