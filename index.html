Azure Cross-Platform Storage Explorer

A cross-platform implementation of azure storage explorer allowing the manipulation of azure blob storage accounts and eventually tables and queues.

If you've ever used Azure Blob storage, you'll notice that there really isn't a blob explorer which will let you browse your assets easily. This project aims to make it easier for you to use Azure blob storage through a cross platform client app.
